# XServer Static Deploy Folder

This directory is self-contained and ready to publish.

Contents:
- index.html / app.js / styles.css
- assets/
- data/

Recommended usage:
1. Push the contents of this directory to the root of a GitHub repository used for XServer Static auto deploy.
2. Or upload this directory's contents directly with XServer file manager / FTP.

Refresh this folder from source with:
- npm run build:xserver-static
